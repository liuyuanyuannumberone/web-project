今天我读了群里面发的“多点规律自己，多宽待他人”    他让我明白啦:不仅要对自己要严格，反而要宽容对待他人。古话说:“己所不欲，勿施于人。”以为自己不喜欢的事物，也不要推卸给别人，否则本来是一件平常的小事，会变成你一生的污点，宽容的对待别人，就等于给自己留余地，甚至别人还会忘记以往你对别人的坏。    诚实的对待他人，你的人格（或品质）才会越来越好。如果你有一次撒谎，以前所有的好事都弥补不了这一个人生的污点，加酿成大祸。要用机智聪明的方法、言语来真诚的告诉对方，自己的不足。    取人之长，补己之短。当你发现别人的不足时，同时思考自己有没有这种地方的缺点；当发现别人的优点时，立刻学习，多多指教，要大胆的表达意见。好探索新知识，这是一个人一个人做出这种品质的基本。    我打算以后这样做:第一，自己所遇到的困难，不要托付给别人。第二，诚实待人，认真做事。第三，多发现别人的优缺点，随时反省自己身上有没有相同点和不同点。 小礼物走一走，来简书关注我 小礼物走一走，来简书关注我 作者：汪彦霖 链接：https://www.jianshu.com/p/752422225e8b 來源：简书 简书著作权归作者所有，任何形式的转载都请联系作者获得授权并注明出处。 content
